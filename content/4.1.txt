Для создания оферты требуется на странице «Мои оферты» нажать на кнопку «Создать оферту» или на ссылку «Создать оферту» в меню раздела «Мои оферты». В результате откроется форма ввода сведений об оферте со статусом «Ввод сведений», в которой по умолчанию заполнены следующие поля с возможностью дальнейшего редактирования):
- цена за единицу продукции с НДС - значение «0,00»;
- ставка НДС - значение «18%»;
- минимальное количество - значение «1»;
- максимальное значение - установлен флажок в поле отметки «Неограниченное количество»;
- дата начала действия - значение «текущая дата»;
- дата окончания действия - значение «текущая дата + 1 год»;
- количество дней для поставки товара/выполнения работы/оказания услуги с момента заключения договора, от: … до: … - значение «0»;
- прикреплен файл проекта договора или других документов, если в разделе «Мои документы» в столбце «По умолчанию» для этих документов установлен флажок.
